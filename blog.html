Question-1: What are the different ways to select an element in the DOM?

There are five way to select DOM Element.

  • 1. getElementById.
  • 2. getElementsByClassName.
  • 3. getElementsByTagName.
  • 4. querySelector.
  • 5. querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1. innerHTML

Some HTML content sets in any HTML Tags. Example: 'p' Tag has some content. We can set this tag inside 'div' or others tag.

2. innerText

Only the visible Text, we can sets or store any HTML Tag.

2. innerContent

It sets the text content of an element. it is same as like innerText but, it is fast from innerText.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where an event listener is placed on a parent element and all events of its child elements are handled through that event. Event delegation work like Event Bubbling.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a event propagation process. It working - Parent to child and get target phase, then return back to child to parent tag.

Question-5: How do you create, add, and remove elements using JavaScript?

1. For create - document.createElement("")

2. For add - appendChild(), prepend(), append() and insertBefore()

3. For remove - remove(), removeChild()

We can use this DOM nodes to create element, add element, remove element.
